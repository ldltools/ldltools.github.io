    LDL Tools

    Linear Dynamic Logic on finite traces (LDLf)1 is a recently-proposed extension of Linear Temporal Logic (LTL)2, a well-known propositional temporal logic introduced decades ago. LDL Tools is a collection of a variety of small programs ranging from those for verifying LDLf formulas to an experimental DSL processor for smart contract development.

    ldlsat

    ldlsat is a SAT solver for LDLf.

    dsl4sc

    dsl4sc is a small experimental domain-specific language for defining correct smart contracts in the sense that they are guaranteed, by construction, to meet their formally-specified requirements. A set of tools are provided for processing smart contracts in dsl4sc and deriving useful results.
